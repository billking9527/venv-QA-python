Q&A

Q: How to create a virtual environment (venv) in Python?

A: To create a virtual environment in Python, use the command: python -m venv venv. This will create a virtual environment named venv in your current directory.

问：如何在Python中创建虚拟环境（venv）？

答：在Python中创建虚拟环境，请使用命令：python -m venv venv。这将在当前目录中创建一个名为venv的虚拟环境。

Q: How to activate the virtual environment?

A: To activate the virtual environment, use the following command:

  • On Windows: .\venv\Scripts\activate
  • On macOS/Linux: source venv/bin/activate

问：如何激活虚拟环境？

答：要激活虚拟环境，请使用以下命令：

  • 在Windows上：.\venv\Scripts\activate
  • 在macOS/Linux上：source venv/bin/activate

Q: How to deactivate the virtual environment?

A: To deactivate the virtual environment, use the command: deactivate. This will return you to the global Python environment.

问：如何停用虚拟环境？

答：要停用虚拟环境，请使用命令：deactivate。这将使您返回到全局Python环境。

Q: Can I use python -m venv venv001 to create a venv called venv001?

A: Yes, you can use the command python -m venv venv001 to create a virtual environment named venv001.

问：我可以使用python -m venv venv001来创建一个名为venv001的虚拟环境吗？

答：是的，您可以使用命令python -m venv venv001来创建一个名为venv001的虚拟环境。
